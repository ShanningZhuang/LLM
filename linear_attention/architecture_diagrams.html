How the neural network blocks are actually wired — not just the math

Standard Transformer Block (Llama-style)
The baseline you already know. A pre-norm block with multi-head softmax attention followed by a SwiGLU MLP. Every token attends to all previous tokens via a full KV cache — O(L²) during training, O(L) KV cache per token at inference.

Mamba Block (S6 / Selective SSM)
Mamba replaces both attention and MLP with a single gated SSM block. The input splits into two paths: one goes through Conv→SSM (the "content" path), the other is a simple SiLU gate. The SSM uses input-dependent B, C, Δ parameters (selectivity). During inference, only a fixed-size state h is maintained — no KV cache.

Key insight: Mamba has no separate MLP block — the gated SSM replaces both attention and FFN. Each Mamba layer is essentially one block, not attention+MLP. The "gate path" plays the role of the GLU gating that would normally be inside an MLP.
Gated DeltaNet Block (Token Mixing)
Gated DeltaNet follows the Llama-style layout: a token mixing layer (Gated DeltaNet replacing attention) followed by a SwiGLU MLP for channel mixing. The token mixing block projects Q, K, V plus two scalar gates (α, β), applies short convolutions and normalization, then runs the gated delta rule recurrence.

Why these components matter:
• Short Conv1d — captures local n-gram patterns (like "New York") that the recurrence alone would miss; enables induction heads in a single layer.
• L₂ Norm on Q, K — when β=1 and ‖k‖₂=1, the matrix (I − kkT) becomes a projection that cleanly erases one direction in memory.
• α gate — global forgetting (topic shifts, document boundaries).
• β gate — surgical update strength (how aggressively to overwrite a specific memory slot).
• Output RMSNorm — replaces the unstable 1/Σ normalizer from original linear attention.

The fundamental tradeoff: Softmax attention stores every past token (perfect recall, expensive). Mamba compresses into a vector h (cheap, but information gets "smudged"). Gated DeltaNet compresses into a matrix S (richer than a vector — can surgically overwrite specific associations — but still fixed size). The matrix state S is essentially an associative memory that maps keys to values.
Hybrid Architecture (e.g., Qwen 3.5)
Production models don't use just one block type. They interleave linear attention layers with a few softmax attention layers in a repeating pattern. A typical ratio is 3:1 — three Gated DeltaNet blocks for every one sliding window attention block.

Why hybrid works best:
• Linear layers (Gated DeltaNet / Mamba) handle the bulk of computation efficiently — O(1) per token at inference, good at tracking slow-changing context.
• Attention layers (sliding window or global) provide exact retrieval over a local window — catching details the compressed state might miss.
• The DeltaNet paper found that adding just 2 global attention layers to a 24-layer model surpassed a full Transformer in retrieval benchmarks.
• Qwen 3.5, Jamba, and Samba all use variants of this pattern in production.
